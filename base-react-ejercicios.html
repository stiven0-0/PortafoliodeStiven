Ejercicios Base React

Map | Filter | Find | Reduce

Ejercicio 1

Uso de map, filter y find para manipular productos.

• Mostrar nombres en mayúsculas • Filtrar productos baratos • Buscar un producto
Código
const productos = [
{nombre:"Laptop", precio:1200},
{nombre:"Mouse", precio:25},
{nombre:"Teclado", precio:45}
];

const nombres = productos.map(p=>p.nombre);
const baratos = productos.filter(p=>p.precio < 50);
const buscar = productos.find(p=>p.nombre==="Mouse");

console.log(nombres);
console.log(baratos);
console.log(buscar);

Ejercicio 2

Analizar inventario usando reduce.

• Calcular total inventario • Filtrar productos • Mostrar resultados
Código
const productos = [
{nombre:"Laptop", precio:1200, cantidad:5},
{nombre:"Mouse", precio:25, cantidad:20},
{nombre:"Teclado", precio:45, cantidad:15}
];

const total = productos.reduce((acc,p)=>{
return acc + (p.precio * p.cantidad);
},0);

console.log(total);

Ejercicio 3

Clasificación de productos usando reduce.

• Calcular total • Mayor stock • Mostrar resultados
Código
const productos = [
{nombre:"Laptop", cantidad:5},
{nombre:"Mouse", cantidad:20},
{nombre:"Teclado", cantidad:15}
];

const mayor = productos.reduce((max,p)=>{
return p.cantidad > max.cantidad ? p : max;
});

console.log(mayor);
